Dark Side of the Moon Base Map

A basemap that celebrates a legendary Pink Floyd Album

This “slippy tileset” basemap is a fully functional web map styled after the famous Pink Floyd album The Dark Side of the Moon. As this assignment required me to think about cartography as an art, I chose to be inspired by the art that I see every day, album covers. By applying the minimalist, bold design of The Dark Side of the Moon album cover to a map and using visual variables to create a pleasing visual hierarchy, I created a map that is both simple to use and fascinating to look at. I used Mapbox Studio to create the basemap and HTML to create the associated explainer.
